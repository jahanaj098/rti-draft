# Kerala RTI Online Drafting Assistant

A web-based tool to help citizens of Kerala draft legally valid RTI applications for local bodies and government authorities.

## ğŸš€ Features

- **Step-by-Step Wizard**: Easy-to-use interface for rural and first-time users.
- **Authority Selection**: Pre-mapped Kerala districts and local body types.
- **Dynamic Questions**: Add up to 10 specific questions with helpful tips.
- **Signature Pad**: Capture signatures via touch, camera, or file upload.
- **Client-Side PDF Generation**: Generates a formatted RTI application (Form A) instantly.
- **Privacy First**: No data is stored on servers; everything happens in the browser.
